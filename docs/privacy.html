Privacy Policy

TabPilot — Window & Tab Manager for Chrome
Last updated: March 22, 2026

Summary

TabPilot does not collect, store, or transmit any personal data. All extension data stays on your device in Chrome's local storage.

Data Collection

TabPilot collects no user data. Specifically:

  • No personally identifiable information
  • No browsing history
  • No page content
  • No cookies or authentication tokens
  • No analytics or telemetry
  • No crash reports

Local Storage

TabPilot stores the following data locally on your device using chrome.storage.local:

  • User preferences (theme, layout, auto-suspend timer settings)
  • Saved session projects (window and tab URLs/titles)
  • AI provider configuration (provider choice and API key, if configured)
  • Custom routing rules for tab grouping

This data never leaves your device unless you explicitly use the export feature, which saves a file to your local filesystem.

AI Feature (Optional)

TabPilot includes an optional AI-powered tab grouping feature. If you choose to use it:

  • You provide your own API key for OpenAI or Anthropic, or use Chrome's built-in AI (no key required)
  • TabPilot sends only tab titles and domain names to the AI provider you configure
  • No page content, cookies, browsing history, or personal data is transmitted
  • API calls go directly from your browser to the provider — TabPilot has no intermediary server
  • Your API key is stored locally and never shared with TabPilot or any third party

You can use TabPilot without the AI feature. No data is sent to any external service unless you explicitly configure an AI provider.

Permissions

TabPilot requests only the Chrome permissions necessary for its features:

  • tabs — read tab titles and URLs to display in the dashboard
  • tabGroups — read and create Chrome tab groups
  • storage — persist settings and saved sessions locally
  • alarms — schedule auto-suspension of inactive tabs
  • sidePanel — render the side panel UI
  • declarativeNetRequest — modify request headers for AI API calls (CORS)

Third-Party Services

TabPilot does not integrate with any third-party analytics, advertising, or tracking services. The only external network requests are to AI API endpoints (OpenAI or Anthropic) when you explicitly configure and use the AI feature with your own API key.

Data Sharing

TabPilot does not sell, share, or transfer any user data to third parties. There is no user data to share.

Children's Privacy

TabPilot does not knowingly collect any information from children under the age of 13.

Changes to This Policy

If this privacy policy changes, the updated version will be published at this URL. The "last updated" date at the top will reflect the revision date.

Open Source

TabPilot is open source. You can audit the complete source code at github.com/ankitksharma/tabpilot.
