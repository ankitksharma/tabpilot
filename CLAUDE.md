## Project "TabPilot" – Window & Tab Manager (Chrome Extension)

## Core capabilities

- **Bird’s-eye window & tab overview** – see every open tab across all Chrome windows in a multi-column dashboard.
- **Drag-and-drop tab control** – reorder tabs inside a window or move them to another/new window with simple drag-and-drop.
- **Tab & window suspension** – suspend every tab in a window (or on restore) to free RAM and cut CPU usage. 
- **Session “Projects”** – save the current set of windows/tabs and reopen later; restored tabs load suspended to avoid heavy reloads.
- **Bulk actions + undo** – select many tabs at once to close, move, mute, or suspend, and instantly undo accidental closes.
- **Power search & sort** – live-filter by title/URL, sort alphabetically or by domain, and group tabs from the same site together.
- **Import / export & cloud sync** – export open or saved windows to CSV/JSON and re-import later; optional cloud backup with a one-time licence.
- **Real-URL visibility for suspended tabs** – still shows the true URL even when another suspender extension hides it.
- **Audio controls** – identify which tab is playing sound and mute/un-mute directly from the manager.
- **Custom UI & shortcuts** – dark theme, multi-column wide-screen layouts, full keyboard navigation, and configurable key bindings.
- **Privacy-first** – no third-party data sharing; requested permissions only cover the listed features.
